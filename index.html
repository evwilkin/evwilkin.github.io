Hi-Lo Guessing Game

A card-based guessing game

Hi-Lo

This is a jQuery-powered version of the classic "Higher/Lower" card-guessing game. I wanted to create a mobile-first attractive experience that utilizes changing graphics and constant feedback to the users. This is a 2-player game, with logic built in to allow you to pass turns to the other player only after scoring 3 correct answers. I utilized the Deck of Cards API (http://deckofcardsapi.com/) for the random card selections and images, and SweetAlerts for the popup boxes.

Technologies:
  • HTML5
  • CSS3
  • Javascript
  • jQuery
  • Deck of Cards API
  • SweetAlert.js
Responsibilities:
  • Front end developer - designed and built entire game
  • Utilized Deck of Cards API for generating new deck of cards for each game and drawing random card for each turn
  • Utilized jQuery for AJAX calls, DOM manipulation, and Event Handlers
  • Implemented SweetAlert.js for more attractive alerts
