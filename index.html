Bankly

Visualize your Spending.

Bankly

A personal finance app built on Ruby on Rails over the course of a week with a team of 4. Users can create a profile in order to log expenses and enter their monthly budget. The app then uses data visualization to give monthly breakdowns of expenses organized by category, as well as gives a full ledger of purchases.

Technologies:
  • HTML5
  • CSS3
  • SASS
  • Javascript
  • Ruby on Rails
  • Bootstrap
  • jQuery
  • Postgres
  • Sequelize
  • Heroku
  • Cloudinary API
  • Chart.js
  • fullPage.js
  • Font Awesome
Responsibilities:
  • Project Manager/GitHub Master
  • Backend Developer - built out all routes delivering all data to the page
  • Built Profile Page
  • Set up and implemented Cloudinary API for image uploads
  • Implemented Chart.js library for data visualization
  • Deployed app to Heroku

Restaurant Week

Seattle Restaurant Week, Spring 2016

Restaurant Week

A web app built to provide listings of all restaurants participating in Seattle Restaurant Week 2016, including information about and links to each restaurant data scraped from the official Restaurant Week website. Coming Soon: search functionality.

This app was built in a day with a team of four for a MEAN-stack hackathon, and indeed uses Mongo, Express, Angular and Node. I set up the entire backend, feeding all the data to the site as well as building and styling the restaurants listings page.

European Honeymoon Photo Gallery

Italy, Greece, and Iceland - Fall 2016.

European Honeymoon Photo Gallery

Erin and I were married in August 2016, and pursuing our love of travel we planned an extended Honeymoon abroad in Europe where we had been separately but never together. This site was built as a way to share our memories with friends and family through a full-page photo album.

In addition to showcasing our travel memories, this project also served as a chance to learn and use fullPage.js, Foundation and HTML video. It was completed for a one-day front end hackathon project, though I look forward to using it as a canvas to continue to experiment on with new technologies.

Ludicrous Laws

Head-to-head tour of America's craziest laws.

Ludicrous Laws

I was inspired to create this game after seeing an article listing some of the weirdest laws in the country that a friend shared with me. I thought it would make a good way to get a kick out of the humor in the laws but also make an entertaining and competitive interactive experience.

This is one of the first games I built, utilizing purely front end technologies including HTML, CSS, Javascript, and jQuery. I coded all the styling for the street signs in pure CSS, and used jQuery for the car animations.
