A public dashboard designed to clearly distribute information collected by the Environmental Protection Agency. Data is organized by category, sortable, and linked to corresponding data tables.   • AnyChart
Responsibilities:
  • Frontend Developer - built UI and data visualizations in Oracle Apex
  • Backend Developer - built queries needed for each chart, including filtering and drill-down options
  • Client contact - collaborated with clients for direct feedback and continuous iteration

Ludicrous Laws

Head-to-head tour of America's craziest laws.

Ludicrous Laws

I was inspired to create this game after seeing an article listing some of the weirdest laws in the country that a friend shared with me. I thought it would be even more fun to get a kick out of the humor in the laws as a head-to-head game providing an entertaining and competitive interactive experience. This is one of the first games I built, utilizing purely front end technologies including HTML, CSS, Javascript, and jQuery.

Technologies:
  • HTML5
  • CSS3
  • Javascript
  • jQuery
  • Bootstrap
  • SweetAlert.js
Responsibilities:
  • Front end developer - designed and built entire game
  • Used pure CSS to design street signs
  • Utilized jQuery for car animations
  • Used vanilla javascript object constructor to code in game trivia for all 50 states
