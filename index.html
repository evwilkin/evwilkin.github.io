Bankly

Bankly

A personal finance app built on Ruby on Rails over the course of a week with a team of 4. Users can create a profile in order to log expenses and enter their monthly budget. The app then uses data visualization to give monthly breakdowns of expenses organized by category, as well as gives a full ledger of purchases.

Technologies:
  • HTML5
  • CSS3
  • SASS
  • Javascript
  • Ruby on Rails
  • Bootstrap
  • jQuery
  • Postgres
  • Sequelize
  • Heroku
  • Cloudinary API
  • Chart.js
  • fullPage.js
  • Font Awesome
Responsibilities:
  • Project Manager/GitHub Master
  • Backend Developer - built out all routes delivering all data to the page
  • Built Profile Page
  • Set up and implemented Cloudinary API for image uploads
  • Implemented Chart.js library for data visualization
  • Deployed app to Heroku

Restaurant Week

  • Cheerio
  • Angular
  • Node
  • Express
  • MongoDB
  • Mongoose
  • Heroku
Responsibilities:

  • Front end developer - built and styled the restaurants listings page
  • Database management and schema design for Mongo database
  • Implemented user authentication using JSON Web Tokens and bcrypt password hasing

Ludicrous Laws

Head-to-head tour of America's craziest laws.

Ludicrous Laws

I was inspired to create this game after seeing an article listing some of the weirdest laws in the country that a friend shared with me. I thought it would be even more fun to get a kick out of the humor in the laws as a head-to-head game providing an entertaining and competitive interactive experience. This is one of the first games I built, utilizing purely front end technologies including HTML, CSS, Javascript, and jQuery.

Technologies:
  • HTML5
  • CSS3
  • Javascript
  • jQuery
  • Bootstrap
  • SweetAlert.js
Responsibilities:
  • Front end developer - designed and built entire game
  • Used pure CSS to design street signs
  • Utilized jQuery for car animations
  • Used vanilla javascript object constructor to code in game trivia for all 50 states
